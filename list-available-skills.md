# list-available-skills

列出当前对话上下文中所有可用的技能。

## 核心功能

从当前对话上下文中解析 `<available_skills>` 标签，提取所有技能的名称和描述，以友好的格式输出。

## 使用场景

- 想要了解当前 Agent 有哪些技能可用
- 需要查找某个功能是否有对应的技能
- 探索系统中已安装的技能能力
- 调试技能加载问题

## 输出格式

技能列表以表格形式展示，包含：
- **序号**：技能编号
- **名称**：技能的唯一标识名
- **描述**：技能的简短功能说明

## 使用方式

在对话中直接请求：

```
列出当前可用的技能
有哪些技能可以使用？
查看 available_skills
```

## 注意事项

- 技能列表来自系统注入的 `<available_skills>` 上下文
- 列表可能因系统配置而被截断（通常限制在 30000 字符内）
- 某些技能可能因缺少凭证或配置而被过滤
- 技能列表在不同 Agent 和会话中可能有所不同
